Al hacer un fork (se guarda en una variable pid_t):
- Si es un valor menor que 0 el fork falló
- Si el valor de retorno del fork es 0 estamos en el proceso hijo
- Si el valor del fork es mayor que 0 es el padre

la isntrucción wair() retorna el PID del hijo que terminó.

Para generar un exit status se hace con las instrucción exit()

PIPES: mecanismo de comunicación entre procesos de la misma máquina
- Es unidireccional
- Ha de haber una relación de jerarquía (padre e hijo) o parentesco (hermanos)
